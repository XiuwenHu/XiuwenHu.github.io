hexo博客搭建

nodeJS的安装与测试

1.在官网下载最新的nodeJS文件，并安装。

2.在终端窗口对安装号的nodeJS进行测试，在桌面位置输入node-v并回车，得到v4.4.5的版本号，说明安装成功。

hexo的安装与测试

1.在桌面位置先创建一个blog的文件夹，在终端窗口中进入blog文件夹，再输入npm install hexo -g然后回车来安装hexo，若出现error，则是因为权限不够，这时输入sudo npm install hero -g 回车，然后会出现输入密码的命令，输入本机密码后便能成功安装。
2.终端窗口输入hexo init 进行对hexo目录的初始化，可以输入hexo new 文件名 来创建新blog。输入hexo server 启动本地服务，可以得到一个本地blog地址，浏览器进入即可看到当前的blog。若要退出此状态，输入Ctrl+C退出。

搭建github在线blog

创建github账号并与本地连接

1.在github官网创建一个自己的账号，记住用户名及邮箱，并通过邮箱认证。
2.在终端窗口中输入git，回车可得当前的git命令及常用用法。接下来进行对ssh-key的配置，输入ssh-keygen，并连续三次回车。再输入vim ~/.ssh/id_rsa.pub，回车得到ssh-key，复制ssh-key，进入自己的github设置，在ssh粘贴ssh-key。
3.接着配置github与本地机器验证，输入git config –global user.name”用户名”，回车输入git config - -global user.email “验证邮箱地址”。
4.在github的new repository新建博客域名仓库，输入用户名.github.io。

blog的底层设置

打开blog文件夹的_config.yml文件，可以对blog的一些基本情况进行设置，比如
title: 新的征程
subtitle: 新的开始
description: 记录学习点滴
author: Sakura
language: zh-CN

最后设置最下面的deploy
type: git
repo: 域名仓库的https地址
其中域名仓库的https地址在自己的github博客的域名仓库中获得。

发布博客上传到线上

1.终端窗口中输入npm install hexo-deployer-git –save,回车对hexo进行git的安装和保存。
2.对编辑好的博客进行hexo server本地测试后，再输入hexo g，回车生成浏览器能识别的html和css文件，再输入 hexo d，回车将本地数据上传到自己的github博客中去。

这样就可以完成自己的博客了。^_^
